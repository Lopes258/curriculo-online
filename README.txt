# Meu Projeto de Currículo Online

Este é um projeto em andamento onde estou criando um currículo online utilizando **Streamlit** e a biblioteca **Astral**. A ideia é construir um website interativo que simula o meu currículo, com informações sobre minhas habilidades, projetos e dados de contato.

## Tecnologias Utilizadas

- **[Streamlit](https://streamlit.io/)**: Uma ferramenta incrível para criar aplicativos web interativos de forma simples e rápida.  
- **[Astral](https://docs.astral.sh/uv/)**: Biblioteca utilizada para criar o website.

## Links Importantes
